How to Use This Dashboard

This interactive dashboard allows you to explore antimicrobial resistance (AMR) surveillance data across European countries.

Filters (Left Panel)
  • Countries: Select which countries to include in your analysis
  • Culture Material: Filter questions by specimen type (blood, urine, etc.)
  • Pathogens: Focus on specific bacteria of interest
  • Resistances: Filter by antibiotic resistance types
Sections (Top Buttons)
  • National Surveillance: General surveillance practices
  • National Genomic Surveillance: Genomic sequencing approaches
  • National Guidance: Guidelines and recommendations
Tabs
  • Map: Visualize survey responses geographically with charts
  • Survey Results: View detailed data in table format and download results
  • Info: Learn more about the project and access legal information

Tip: Use the 'Reset filters' button to return to the default view at any time.
