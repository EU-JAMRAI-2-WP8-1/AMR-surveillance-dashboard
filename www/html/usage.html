This dashboard presents current national AMR surveillance information in humans from 28 of the 30 EU-JAMRAI 2 partner countries. It is designed as a tool for national and international AMR experts and policymakers to explore, compare, and analyse surveillance information across Europe.

For project details and contacts, please visit the Info tab.

How to Use the Dashboard

  1. Select a question from the dropdown menu.
  2. The map shows answers by country.
  3. The bar chart shows combined answers for all countries.

Customize Your View (Left Panel)

Use the tools below to focus on what matters to you:

Tabs

  • Graphics: View answers on a map and bar chart.
  • Table: See answers in an interactive table (filters apply here too - see here below).

Sections

Questions are grouped into three categories:

  • National Surveillance: How AMR is monitored at the national level (e.g., which pathogens are under mandatory surveillance?).
  • National Genomic Surveillance: How genomic sequencing is used for AMR (e.g., which pathogens are analysed using WGS?).
  • National Guidance: Guidelines that use AMR data or influence surveillance (e.g., national treatment guidance for common infections).

Filters

Narrow your selection by:

  • Countries: Show all or only specific countries.
  • Culture material: Filter by specimen type (e.g., blood, urine).
  • Pathogens: Focus on specific bacterial pathogens.
  • Resistances: Select specific antibiotic resistances.

Tips for Better Navigation

  • Use filters to quickly find relevant questions.
  • Click "Reset filters" to return to the default view.
  • Download data in the Table tab.
  • Click the instructions button (top left) to revisit this guide.
